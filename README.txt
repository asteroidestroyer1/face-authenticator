# Face Authentication System using OpenCV

This project implements a real-time Face Authentication System using OpenCV DNN and LBPH.

## Features
- Face registration using webcam
- Face detection using DNN
- Face recognition using LBPH
- Real-time authentication

## Tech Stack
- Python
- OpenCV
- NumPy

## How to Run
pip install -r requirements.txt  
python main.py
